# licenses-aggregator

Aggregate license notes of OSS libraries used in your Node.js project

## Installation

```sh
npm install licenses-aggregator
```

## Usage


```js
import { multiply } from 'licenses-aggregator';

// ...

const result = await multiply(3, 7);
```
